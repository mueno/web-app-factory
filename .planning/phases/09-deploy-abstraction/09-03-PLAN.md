Create `tools/deploy_providers/gcp_provider.py` (replacing the placeholder stub from Plan 01):

1. `_check_gcloud_auth() -> tuple[bool, str]`: Pre-flight checks in order:
   a. Check gcloud CLI on PATH: subprocess.run(["gcloud", "--version"], capture_output=True, text=True, timeout=10)
      - FileNotFoundError: return (False, "gcloud CLI not found. Install from: https://cloud.google.com/sdk")
      - Non-zero exit: same error
   b. Check valid auth: subprocess.run(["gcloud", "auth", "print-access-token"], ..., timeout=15)
      - Non-zero: return (False, "GCP auth token invalid or expired. Run: gcloud auth login")
   c. Check project configured: subprocess.run(["gcloud", "config", "get-value", "project"], ..., timeout=10)
      - Empty or "(unset)": return (False, "No GCP project configured. Run: gcloud config set project <PROJECT_ID>")

2. `_get_gcloud_region() -> str | None`: Read gcloud config get-value run/region. Return None if empty. This is optional pre-flight info, not blocking.

3. `_GCP_URL_RE = re.compile(r"Service URL:\s+(https://[^\s]+\.run\.app)")`

4. `_extract_gcp_url(stderr: str) -> str | None`: Search stderr for the URL pattern.

5. `GCPProvider(DeployProvider)`:
   - `deploy(self, project_dir, env) -> DeployResult`:
     - Call _check_gcloud_auth(); if fails, return DeployResult(success=False, ...)
     - Read nextjs_dir from env.get("nextjs_dir") or str(project_dir)
     - Derive service_name from env.get("app_name", "web-app-factory-app")
     - Build cmd: ["gcloud", "run", "deploy", service_name, "--source", ".", "--allow-unauthenticated", "--quiet"]
     - If region available from _get_gcloud_region(): append ["--region", region]. Otherwise append ["--region", "us-central1"] and log warning about using default region.
     - subprocess.run(cmd, cwd=nextjs_dir, capture_output=True, text=True, timeout=600, env={**os.environ})
     - Parse stderr for URL via _extract_gcp_url()
     - Return DeployResult with success/url/provider="gcp"/metadata
   - `get_url(self, result)`: return result.url or raise ValueError
   - `verify(self, url)`: return run_deployment_gate(url).passed

6. SECURITY: All subprocess calls use list args (no shell=True), env={**os.environ}.

7. Add all behavior tests to `tests/test_deploy_providers.py` in a `TestGCPProvider` class. Mock all subprocess.run calls -- no real gcloud execution in tests. Update registry test to remove xfail from test_registry_gcp.
