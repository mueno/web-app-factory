<objective>
Fix the Phase 2a-to-2b project directory handoff and wire GovernanceMonitor into the live pipeline runner.

Purpose: Phase 2b currently passes the pipeline root directory (ctx.project_dir) to the build agent and build gates, but the Next.js project lives at ctx.project_dir.parent / ctx.app_name (a sibling directory created by create-next-app in Phase 2a). This means the build agent writes code to the wrong location, and the build gate runs npm build / tsc in a directory without package.json. Additionally, GovernanceMonitor is fully implemented but never instantiated in contract_pipeline_runner.py, leaving the live pipeline with zero phase-skip enforcement.

Output: Fixed phase_2b_executor.py, updated contract_pipeline_runner.py with nextjs_dir gate parameter and GovernanceMonitor wiring, and updated/new test assertions.
