<objective>
Create Supabase Auth TypeScript templates (middleware, login, signup, signout, callback), an AUTH_SETUP.md README template for manual OAuth setup, and a Python renderer that copies them into generated apps.

Purpose: These templates are the foundation of the auth scaffolding — every generated app with Supabase gets cookie-based session management, OAuth sign-in pages, protected route redirects, and a step-by-step README for Google Cloud Console and Apple Developer Portal setup. Templates must follow the 2026-03 Supabase SSR best practices: `getUser()` (not `getSession()`), anon key in middleware (not service_role), custom OAuth buttons (not the archived `@supabase/auth-ui-react`).

Output: 5 TypeScript template files + 1 Markdown README template + 1 Python renderer module + 2 test files
