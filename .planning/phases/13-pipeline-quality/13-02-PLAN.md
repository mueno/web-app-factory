<objective>
Add an E2E Playwright gate that validates form submission flows in generated Next.js apps. The gate starts `next start`, navigates to the form page discovered from screen-spec.json, fills and submits the form, and asserts the result page renders content.

Purpose: The FLOW-01 static analysis gate catches parameter name mismatches at the source level, but cannot detect runtime failures (JS errors, hydration issues, navigation failures). The E2E gate catches what static analysis misses by actually running the form flow in a real browser.

Output: New `e2e_gate.py` gate executor, YAML contract update, gate dispatch wiring, and comprehensive unit tests.
</objective>

```python
def run_e2e_gate(
    project_dir: str,          # absolute path to Next.js project (nextjs_dir)
    spec_path: str,            # absolute path to screen-spec.json
    phase_id: str = "2b",
) -> GateResult:
```

Implementation details:

1. **Import guard:** `try: from playwright.sync_api import sync_playwright / except ImportError: sync_playwright = None`. If None, return `GateResult(passed=False, gate_type="e2e_form_flow", issues=["playwright is required but not installed"])`.

2. **Load screen-spec.json:** Parse JSON. If file missing or malformed, return `GateResult(skipped=True, skip_allowed=True)`.

3. **Discover form/result route pair:**
   - `_find_form_route(spec)`: Iterate screens. A screen is a "form page" if any component name contains case-insensitive: "Form", "Input", "Search", "Calculator", "Simulator", "Builder", "Creator", "Editor". Return the first matching screen's route.
   - `_find_result_route(spec, form_route)`: Find a screen whose route contains "result", "output", "summary", "report", or is the screen immediately after the form screen in the screens list. Return route or None.
   - If no form route found: return `GateResult(skipped=True, skip_allowed=True, gate_type="e2e_form_flow", issues=["No form-like components found in screen-spec.json — E2E form flow gate skipped"])`.
   - If no result route found: return `GateResult(skipped=True, skip_allowed=True, gate_type="e2e_form_flow", issues=["Form page found but no result route identified — E2E form flow gate skipped"])`.

4. **Start `next start` server:**
   - Find free port: `socket.socket(); s.bind(("", 0)); s.getsockname()[1]`
   - Set `env = {**os.environ, "PORT": str(port)}` (Pitfall 1: use PORT env var, not --port flag)
   - Start: `subprocess.Popen(["npm", "run", "start"], cwd=project_dir, env=env, stdout=subprocess.PIPE, stderr=subprocess.STDOUT, start_new_session=True)`
   - Wait for ready: `threading.Event` + background thread matching `_READY_RE = re.compile(r"(?:ready|Ready|started server)", re.IGNORECASE)`. Timeout 30s.
   - If timeout: kill process, return `GateResult(passed=False, issues=["next start did not become ready in 30s"])`

5. **Run Playwright form flow:**
   - `with sync_playwright() as pw:`
   - `browser = pw.chromium.launch(headless=True)`
   - Navigate to `http://localhost:{port}{form_route}`, `wait_until="networkidle"`, timeout 30000ms
   - Find input fields: `page.locator("input:visible").all()` — fill each with test data ("test", "test@example.com", "100" etc based on input type attribute)
   - Click submit: `page.locator("button[type='submit'], button:has-text('Submit'), button:has-text('Calculate'), button:has-text('Search'), button:has-text('Go')").first.click()`
   - Wait for navigation: `page.wait_for_url(f"**{result_route}**", timeout=10000)` — catch TimeoutError and return passed=False
   - Assert result page content: `content = page.text_content("body")` — if content and len(content.strip()) > 50, passed=True
   - `browser.close()` in finally block

6. **Server cleanup:** Always terminate server process in a `finally` block using `os.killpg(os.getpgid(proc.pid), signal.SIGTERM)` (same pattern as _dev_server.py). Catch ProcessLookupError.

7. Return `GateResult(gate_type="e2e_form_flow", phase_id=phase_id, passed=True/False, ...)`.

The function must NOT use `async_playwright` (Pitfall from RESEARCH.md: pipeline is synchronous). Use `sync_playwright` exclusively.
